# Caldo Freddo Frontend

React + Vite app that consumes the Django REST API. The UI remains fully decoupled from Django and does not use Django templates.

## Environment

Create `frontend/.env` from `.env.example` when needed:

```env
VITE_API_BASE_URL=http://127.0.0.1:8000/api
```

## Commands

```powershell
npm install
npm run dev
npm run build
npm run lint
```
